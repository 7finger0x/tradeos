# Feature Specification: Market Open Briefing MVP (Phase 4)

**Feature Branch**: `004-market-briefing-mvp` | **Created**: 2026-06-28

## User Stories

### P1 — Generate Pre-Market Briefing
As a trader, I receive a focused market plan with index context, events, watchlist, and risk overlay.

### P1 — Watchlist
As a trader, I see ranked tickers with reasons based on momentum, catalysts, and my historical edge.

### P2 — Briefing History
As a trader, I can review saved briefings from prior sessions.

## Exit Criteria

- [x] Mock market data adapter (swappable)
- [x] Economic + earnings events stored
- [x] Deterministic briefing generator with risk overlay
- [x] Watchlist scoring
- [x] Briefing UI + history
